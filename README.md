# mailu-sogo-grommunio

These files are part of the blog post material for:

- https://weisser-zwerg.dev/posts/groupware-sogo/
- https://weisser-zwerg.dev/posts/groupware-grommunio/

Subdirectories:

- `sogo/`: generator and example env files for the SOGo sidecar setup.
- `grommunio/`: generator, templates, split-delivery inventory, and Mailu-side
  override sources for the grommunio split-delivery setup.
