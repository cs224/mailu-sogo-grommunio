# SOGo Files

This directory contains the public, anonymized files associated with:

- https://weisser-zwerg.dev/posts/groupware-sogo/

Files:

- `generate_sogo_setup.py`: generates the host-side SOGo Compose tree.
- `sogo-via-vps.env`: production-oriented example input for a VPS Traefik path.

All domains, hostnames, mailbox names, and credentials here are anonymized.
